# Glossary

## Deal Types

| Term | Meaning |
|---|---|
| **Guarantee vs %** | Artist takes whichever is greater. Generally assumed to be after expenses unless contract spells out explicitly. |
| **PLUS** | Guarantee + bonus %, after expenses. |
| **GBOR** | Gross Box Office Receipts — total revenue before any deductions. |
| **NBOR** | Net Box Office Receipts — revenue after expenses (taxes, fees, venue expenses, etc). |
| **PP** | Promoter Profit. |
| **Door** | Can be gross or net (varies by contract). Basically same as "vs from dollar 1." |

## Touring / Production Terms

| Term | Meaning |
|---|---|
| **Build** | Time needed to set up lights, lasers, etc. "We need more time for the light and laser build." |
| **CAT5/6** | Ethernet cable. Connects the music laptop (Ableton, on stage) to the VJ laptop (Touch Designer/Resolume, at FOH). Can run our own line but better if venue has it. |
| **Down Stage** | Front of the stage (closest to the audience). |
| **Drop** | Connection point for a cable type. "Where they drop the cable." |
| **Ethernet** | Same as CAT5/6. Needed for syncing music to video — sends musical info from stage to VJ laptop at FOH. |
| **Floor Package** | Portable lights that a band brings (or rents for the night). Usually rest on the stage floor. |
| **FOH** | Front of house — where the mix engineer and VJs are positioned. |
| **HDMI** | Cable connecting a laptop to a video screen. Needs a drop at FOH to connect the VJ laptop. No HDMI drop at FOH = no video. |
| **Hospo** | Hospitality. |
| **Line** | A run of cable. "Is there an ethernet line running from FOH to stage?" |
| **ROS** | Run of show — the schedule for the day. |
| **Splitter** | Similar to a drop — a place to plug into. |
| **Up Stage** | Back of the stage (away from the audience). |

## Standard Requirements

- **Load-in/Soundcheck**: 3 hours from load-in to end of soundcheck.
- **Set finish**: At least 20 minutes before final curfew (for merch sales).
